# Audio Files Required

This folder needs the following audio files. Download from free sources listed below.

## Recommended Free Sound Sources
- **Freesound.org** (CC0/CC-BY) - Best for SFX
- **Pixabay.com/sound-effects** - Free commercial use
- **Kenney.nl/assets** (CC0) - Great for game audio
- **Mixkit.co/free-sound-effects** - Music and SFX
- **OpenGameArt.org** - Game-specific sounds

## Quick Start

1. Visit Freesound.org or Pixabay.com
2. Search using terms above
3. Download as MP3
4. Place in appropriate folder
5. Ensure filenames match exactly

## Audio Format
- Format: MP3 (44.1kHz, 128-320kbps)
- Keep files small (<500KB each for SFX)
- Music can be larger (1-5MB)
